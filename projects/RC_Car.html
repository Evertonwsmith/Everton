Arduino RC Car

A remote-controlled car project featuring motor control and wireless IR communication.

Link to Tinkercad Simulation

Project Overview

This personal project was aimed at creating a 4 motor Infrared Receiver Remote Controlled Car. The system uses an Arduino Uno to control components, 2 L293D Quadruple Half-H Drivers and an IR Receiver for wireless input. The RC Car features speed control, basic directional controls, and peripheral lighting.

Hardware Specifications

The system is built around the Arduino Uno microcontroller. The following components are integrated as shown in the provided wiring diagram and schematic:

Component List & Connectivity

Component Description

System Architecture & Wiring

The vehicle's logic is distributed across digital and analog pins to manage high-current motors and signal processing simultaneously.

Motor Control Pinout

The motors are controlled via the L293D driver to allow for bi-directional movement and PWM-based speed control.

Motor Position Power (PWM) Input 1 Input 2
Front Left Pin 5 Pin 15 (A1) Pin 14 (A0)
Back Left Pin 6 Pin 4 Pin 7
Front Right Pin 10 Pin 8 Pin 12
Back Right Pin 9 Pin 18 (A4) Pin 19 (A5)
  • IR Receiver: Digital Pin 2
  • Headlights: Pin 17 (A3)
  • Taillights: Pin 16 (A2)

Implementation

The firmware is written in C++ using the Arduino framework and the IRremote.h library for signal decoding.

State Machine Logic

The code utilizes an Enumerated Type (RobotState) to manage the vehicle's behavior. This prevents conflicting commands and ensures smooth transitions between movements.

State Action IR Command Code
STATE_IDLE All motors stop; Taillights ON. 5
STATE_FORWARD All motors move forward; Headlights ON 1
STATE_BACKWARD All motors move backward; Taillights ON 9
STATE_LEFT Left wheels reverse, Right wheels forward 6
STATE_RIGHT Right wheels reverse, Left wheels forward 4
STATE_SPEED_UP Increments MOTOR_SPEED by 10 (Max 255) 10
STATE_SLOW_DOWN Decremets MOTOR_SPEED by 10 (Min 0) 8
STATE_EMERGENCY Flashing lights; Motors stopped 12
