Boolean Algebra Quiz

This project uses 3 input buttons to answer A, B and C to evaluate boolean expressions.

Project Overview

This Arduino-based project creates an interactive boolean algebra quiz using 3 input buttons to answer A, B and C to evaluate boolean expressions.

Features

  • Interactive boolean expression evaluation
  • Three-button input system (A, B, C)
  • Real-time feedback

Technical Details

  • Arduino microcontroller
  • LED indicators for feedback
  • Custom boolean logic implementation

Implementation

The project demonstrates practical application of boolean algebra concepts through hands-on interaction, making abstract mathematical concepts more tangible and engaging for learners.
